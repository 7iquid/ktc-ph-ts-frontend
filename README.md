# How to run:

	* requirements nodejs 'https://nodejs.org/dist/v16.15.1/node-v16.15.1-x64.msi' 

	* git clone: 'https://github.com/7iquid/ktc-ph-ts-frontend.git'

	* ```cd ktc-ph-ts-frontend```
	* ```npm install```
	* ```npm start```
	* enjoy
